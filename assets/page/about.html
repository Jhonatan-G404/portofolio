About

Hallo, perkenalkan nama saya Jhonatan Guritno, saya berasal dari Lampung. Saat ini saya sedang berkuliah di Semarang, di Universitas AKI dengan jurusan Teknik Informatika. Saya sangat menyukai pemrograman, dan keinginan saya adalah menjadi seseorang yang ahli dalam web development. Setelah lulus dari universitas, saya berencana untuk memperdalam pengetahuan tentang berbagai bahasa pemrograman, serta mempelajari framework seperti Laravel dan React. Selain itu, saya juga ingin terlibat dalam proyek-proyek open source untuk meningkatkan keterampilan dan pengalaman saya. Saya percaya bahwa dengan ketekunan dan kerja keras, saya bisa mencapai impian saya menjadi seorang web developer yang handal. Selain menyukai pemrograman, saya juga tertarik pada desain UI/UX, karena saya percaya bahwa tampilan yang menarik dan pengalaman pengguna yang baik sangat penting dalam pengembangan aplikasi. Saya berharap bisa menggabungkan kedua minat ini untuk menciptakan solusi digital yang tidak hanya fungsional tetapi juga menarik bagi pengguna.

"Ketekunan dan kreativitas adalah kunci untuk mencapai impian saya sebagai web developer."
